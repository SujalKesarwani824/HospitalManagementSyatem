
# Hospital Management System (Backend)

A **Hospital Management System backend** built with **Java, Spring Boot, Spring Data JPA, JPQL, and PostgreSQL**.  
The project demonstrates backend development using **JPA, entity relationships, and custom JPQL queries**.

---

## Key Features
- Developed backend to practice **Spring Data JPA** concepts  
- Designed entity relationships for **patients** and **doctors**  
- Performed database operations using **JpaRepository** and custom JPQL queries  
- Hands-on experience with **ORM concepts** and entity mapping  

---

## Tech Stack
- Java, Spring Boot  
- Spring Data JPA, JPQL  
- PostgreSQL  

---

## Notes
- Backend only, no frontend implemented  
- Security credentials are placeholders for testing
